🧹 Chore Tracker (Kid's App)

• Kid checks off chore → disappears from dashboard → you get Telegram message

• Reply yes → money added to balance, chore stays done

• Reply no → chore reappears for kid to redo, no money added

• 📊 Prediction shows monthly/yearly earnings — all chores shown even if not due today

• 🎯 Goals — kid sets savings goals and tracks progress

• 🌤️ Mood — kid logs daily mood

• 📝 Notes — kid sends you a message from the app


⚡ Super Panel (Your App)

• Overview — weekly points + Super Button

• Approvals — approve/reject chores and kung fu

• Chores — add chores, set intervals, make chores reappear

• 💰 Money — track earnings, deduct, mark as paid

• PC — screen time, remote control (only if computer enabled)

• 📝 Notes — see notes your kid sent you


🤖 Telegram Commands

• yes — approve chore + add money

• no — reject chore, reappears for kid

• excused — excuse kung fu absence

• not excused — +100 points

• pending — see all pending approvals

• balance — check money owed


⚡ Points & Super Button

• Missed chore = +50 pts · Missed kung fu = +100 pts

• 100 pts = 1 hour removed per day next week

• Press Super Button at end of week to apply

• Points reset weekly automatically
